# kotlin-blackjack
## 기능 요구 사항
## 입출력
### 입력
 - [x] 참여 이름 (ex. "게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)")
 - [x] 플레이어는 게임을 시작할 때 배팅 금액을 정해야 한다.
 - [x] 추가 분배 여부 (ex. "pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)")
### 출력
 - [x] 초기 분배 출력 (ex. "딜러와 pobi, jason에게 2장의 나누었습니다.")
 - [x] 특정 참가자 카드 목록 출력 (ex. "pobi카드: 2하트, 8스페이드")
 - [x] 딜러 카드 추가 여부 출력 ("딜러는 16이하라 한장의 카드를 더 받았습니다.")
 - [x] 특정 참가자 결과 출력
 - [x] 특정 참가자 수익 출력

## 도메인

### 덱
 [x] 52개의 카드를 shuffle 해서 초기화
 [x] 분배 (pop)
### 카드 (Value Object)
 [x] 모양 / 숫자 / 점수
 [x] Ace의 점수는 1 혹은 11이 될수 있다.

### 참가자 (Abstract Class)
 [x] 플레이어 이름
 [x] 카드목록
 [x] 카드추가 (abstract)

### 배팅 금액
 [x] Money

### Player
 [x] 카드추가
 [x] 배팅금액추가

### Dealer
 [x] 카드추가

### Result (object class)
 [x] 점수 추출
 [x] 수익 추출



## 실행 결과
- [x] 게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
  - pobi,jason
- [x] pobi의 배팅 금액은?
  - 10000
- [x] jason의 배팅 금액은?
  - 20000
- [x] 딜러와 pobi, jason에게 2장의 나누었습니다.
  - [x] 딜러: 3다이아몬드
  - [x] pobi카드: 2하트, 8스페이드
  - [x] jason카드: 7클로버, K스페이드
- [x] pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
  - y
  - pobi카드: 2하트, 8스페이드, A클로버
- [x] pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
  - n
- [x] jason은 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
  - n
  - jason카드: 7클로버, K스페이드

- [x] 딜러는 16이하라 한장의 카드를 더 받았습니다.

- 딜러카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20
- pobi카드: 2하트, 8스페이드, A클로버 - 결과: 21
- jason카드: 7클로버, K스페이드 - 결과: 17

## 최종 수익
- 딜러: 10000
- pobi: 10000
- jason: -20000
